# Key-Events

Key-Events sind messbare Beobachtungen, die im Spiel oder Training gezählt werden können. Sie helfen dem Trainerteam, den Fortschritt bei einzelnen Methoden sichtbar zu machen.

Jedes Key-Event gehört zu einer [[Katalog/Methoden/_index\|Methode]] und enthält eine Aussage mit `_`-Platzhalter für die Anzahl.

> [!info] Hinweis
> Key-Events sind optional und können schrittweise eingeführt werden. Nicht jede Methode hat ein Key-Event.
